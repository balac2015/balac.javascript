    1、代码转换: TypeScript 编译成 JavaScript、SCSS,LESS 编译成 CSS.

    3、模块合并：在采用模块化的项目里会有很多个模块和文件，需要构建功能把模块分类合并成一个文件。

    4、自动刷新：监听本地源代码的变化，自动重新构建、刷新浏览器。
